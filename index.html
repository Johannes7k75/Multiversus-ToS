Outside of the above cases, the data will not be used and the data will not be shared with any third party.

Stored information

The bot only saves data when you give it to it. No data will be saved automatically!
The following data is stored locally.

Set command. If the users sets the channel where the bots reacts!

  • guildId - the id of the guild/server
  • channelId - the id of the channel

Link command. If the users links a username from the game with his account.

  • userId - the id of the user
  • account_id - the account id of the user from the game (gets fetched from the game api)
No other information is stored other than that mentioned above.

The data can be updated when using certain commands. Updating data requires the input of an end user.
